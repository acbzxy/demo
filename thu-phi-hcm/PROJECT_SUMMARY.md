## 🚀 Cách Chạy Dự Án

### 1. Cài đặt Node.js
```bash
# Tải từ https://nodejs.org/
# Yêu cầu: Node.js >= 16.0.0
```

### 2. Cài đặt dependencies
```bash
cd thu-phi-hcm
npm install
```

### 3. Chạy development server
```bash
npm run dev
```

### 4. Mở browser
```
http://localhost:3000
```
